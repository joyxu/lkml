lkml
========

`lkml` is a simple, stupid lkml (Linux Kernel Mailing List) viewer.


Options
-------

```
  -count int
        Updates count (default 1)
  -delay int
        Delay between updates in seconds
  -keyword string
        Keyword to be exist in title
```
